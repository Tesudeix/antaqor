// 08_BRAND_RULES
DO / DON'T

→ ANTAQOR ХИЙНЭ

  • Шууд ярь — цэгтэй, тодорхой, утгатай
  • Data ашигла — тоо, баримт, систем
  • Алдаагаа хэл — wound → weapon arc
  • Маргаан бүтээ — hot take, эрс байр суурь
  • Бүтээ, харуул — ярихаас илүүтэй хий
  • Dark humor ашигла — sarcasm, wit
  • Системийг задал — "яаж ажилладаг" хариулт
  • Visual тогтвортой барь — нэг system дотор
  • Монгол + English холи — code-switch natural

✗ ANTAQOR ХИЙХГҮЙ

  • Зөөлрөхгүй — "магадгүй" хэрэглэхгүй
  • Corporate voice биш — хүйтэн, формал хэл үгүй
  • Цагаан суурь биш — бүх visual dark
  • Олон өнгө биш — зөвхөн orange + cyan + dark
  • Хуулахгүй — бусдын стиль copy хийхгүй
  • Уучлалт гуйхгүй — алдаа хүлээн зөвшөөрч, засна
  • Motivational quote биш — system + action зөвхөн
  • Бүх хүнд таалагдах гэхгүй — polarize
  • Emoji хэт олон биш — max 1-2 per пост
